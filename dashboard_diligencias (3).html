Conectar con Google Sheets

Para conectar el dashboard con tus datos en vivo, necesitas crear un pequeño script en Google Apps Script que sirve como puente. Es un proceso de 2 minutos y solo se hace una vez.

1
Abre tu hoja de Google Sheets y ve a Extensiones → Apps Script
2
Borra todo el contenido y pega este código exacto:

function doGet(e) { var id = '2PACX-1vTCw0z4KyCiP7Wb673xvaIbWeNUp9SYf_q4Z5ZAx34AF8AXBASUG2yzve4NnkYs5zASlIWfbASlBTf7'; var url = 'https://docs.google.com/spreadsheets/d/e/' + id + '/pub?output=csv&_=' + new Date().getTime(); var response = UrlFetchApp.fetch(url); var csv = response.getContentText('UTF-8'); return ContentService.createTextOutput(csv) .setMimeType(ContentService.MimeType.TEXT); }
3
Haz clic en Implementar → Nueva implementación → Tipo: Aplicación web → Ejecutar como: Yo → Acceso: Cualquier usuario → Implementar
4
Copia la URL de la aplicación web que te da Google y pégala abajo
Empieza con https://script.google.com/macros/s/...
